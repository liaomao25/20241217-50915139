股票資訊 API

此 API 提供台灣股票每日交易數據，包括股票代碼、成交量、收盤價等資訊。適合用於金融分析或即時股市追蹤。

查看股票資訊
氣象資訊 API

氣象站資料 API 提供全台氣象站即時天氣數據，包括溫度、濕度、風速、風向及天氣現象。適用於天氣應用或氣象分析。

查看氣象資訊
空氣品質 API

此 API 提供全台各地的即時空氣品質數據（AQI），包括 PM2.5、PM10、主要污染物等，適合用於環境監測應用。
